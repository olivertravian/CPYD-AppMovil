# CPYD-AppMovil
- Proyecto aplicación movil como frontend de la api rest.

## ✨ 1. Instalación
- Instalar librerias con:
``` bash
$ flutter pub get
# or
$ flutter packages get
```
- Se uso como base de pruebas:
  - Pixel 6 Pro
  - Api 33
